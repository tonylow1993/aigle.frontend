### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/tonylow1993/aigle.frontend
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
   or
   ```sh
   yarn add
   ```
   
3. Start the project
    ```sh
   npm start
   ```
   or
   ```sh
   yarn start
   ```
